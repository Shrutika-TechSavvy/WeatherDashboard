# Weather Dashboard

A simple and visually appealing weather dashboard that displays current weather information for a specified city. Built using HTML, CSS, and JavaScript, and powered by the OpenWeatherMap API.Project is consideredd as Mini-project representing the Use of API.

## Features

- Search for current weather information by city name
- Display of city name, temperature, humidity, wind speed, and weather description
- Visually appealing design with background image and weather icons
